# نظام لجنة القبول والتسجيل - ثانوية عمورية V2

نسخة تعمل على GitHub Pages + Firebase Authentication + Realtime Database.

## الدخول
- البريد: admin@amoria.com
- كلمة المرور: التي أنشأتها في Firebase

## الرفع
ارفع كل الملفات والمجلدات إلى المستودع، ثم فعّل GitHub Pages من Settings > Pages.

## الملفات
- index.html: تسجيل الدخول
- dashboard.html: لوحة التحكم
- students.html: إدارة الطلاب
- letter.html: خطاب القبول
- settings.html: إعدادات النظام
- firebase-config.js: بيانات Firebase
- css/style.css: التنسيق
- js/app.js: وظائف النظام
